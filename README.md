# SanofiFollowUpTreamentEmail


This repository contains a **table-based HTML email** (`index.html`) for a REZUROCK® (belumosudil) patient communication. The template follows common transactional/marketing email conventions (nested tables, link-driven CTAs, and text-first layout) and includes regulatory and safety sections required for pharmaceutical communications.

> **Audience:** Patients (US residents) **12 years and older** with chronic GVHD after ≥2 prior systemic treatments.
